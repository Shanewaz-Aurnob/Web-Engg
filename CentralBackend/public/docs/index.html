CSECU Web Server User Manual

Welcome to the CSECU web server user manual. This guide will help you access the server, interact with its resources, and use its APIs effectively.

Base URL

All requests to the server should start with the following base URL:

http://bike-csecu.com:5000/

API Endpoint

The main endpoint for API access is:

http://bike-csecu.com:5000/api/{endpoint}

Static Resources

The server hosts static resources at the following paths:

  • Files: /upload
  • Images: /upload/image

Example Resources

  • File: example.cpp
  • Image: test.webp

Accessing Resources

Fetching Files and Images

Example 1: Accessing a File

To access a file like example.cpp, navigate to:

http://bike-csecu.com:5000/upload/example.cpp

Example 2: Accessing an Image

To view an image like test.webp, navigate to:

http://bike-csecu.com:5000/upload/image/test.webp

Accessing the /teacher API Endpoint

To fetch data from the /teacher endpoint, you can use tools like Postman, Thunder Client, or the Fetch API.

Example Response

{
  "total_records": 11,
  "total_data": 10,
  "data": [

  ],
  "page_count": 2,
  "first_page": 1,
  "next": { "page": 2, "limit": 10 },

Using API Endpoints

Using Postman or Thunder Client

  1. Create a GET Request
    • URL: http://bike-csecu.com:5000/api/teacher
    • Method: GET
  2. Send the Request
    • Click Send and observe the response.

Using Fetch API

To fetch data or download resources using the Fetch API, you can use the following examples:

Example: Fetching API Data


// Fetch data from the /teacher endpoint
fetch('http://bike-csecu.com:5000/api/teacher')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error fetching data:', error));

Example: Downloading a File


// Fetch and download a file
fetch('http://bike-csecu.com:5000/upload/example.cpp')
  .then(response => response.text())
  .then(data => {
    const blob = new Blob([data], { type: 'text/plain' });
    const url = URL.createObjectURL(blob);
    const a = document.createElement('a');
    a.href = url;
    a.download = 'example.cpp';
    document.body.appendChild(a);
    a.click();
    URL.revokeObjectURL(url);
  })
  .catch(error => console.error('Error fetching file:', error));

Example: Downloading an Image


// Fetch and download an image
fetch('http://bike-csecu.com:5000/upload/image/test.webp')
  .then(response => response.blob())
  .then(blob => {
    const url = URL.createObjectURL(blob);
    const a = document.createElement('a');
    a.href = url;
    a.download = 'test.webp';
    document.body.appendChild(a);
    a.click();
    URL.revokeObjectURL(url);
  })
  .catch(error => console.error('Error fetching image:', error));
