Login Endpoint

URL:

http://bike-csecu.com:5000/api/login

Method:

POST

Request Body:

JSON

                {
                    email: string, // Not nullable. Must follow standard email format.
                    password: string // Not nullable. Minimum 8 characters
                }

Authorization:

None

Content Type:

application/json

Response:

JSON

                {
                    "message": string,
                    "data": {
                        "uid": number,
                    }
                }
