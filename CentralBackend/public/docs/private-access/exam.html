Endpoint: /exam

Access information about exams including details about specific exams and filtering options.

Available Endpoints

1. GET /exam/

Retrieve a list of all exams. Data can be filtered using query parameters.

  • Query Parameters:
    • page: Page number for pagination (optional)
    • limit: Number of results per page (optional)
    • sort: Sort exams by a specific field (optional)
  • Response:
    • Success (200): Returns a paginated list of exams.
    • Internal Server Error (500): If there's a server-side issue.

GET /exam/

Request:

fetch('/api/exam', {

})

Additional Information

The /exam endpoint provides access to exam records, including information such as the exam title, associated department, academic session, and date. Use query parameters like page and limit for pagination and sort for sorting results. Contact support for any issues related to exam endpoints.
