Certificate Withdrawal Endpoints for Other Stakeholders

Endpoints to manage searching and posting data for other stakeholders, including retrieval, and submission.

Below are the available endpoints:

  • GET /certificate-withdrawal-otherStakeholders/search-formData/:user_id?pageLimit=${pageLimit}¤tPage=${currentPage}: Retrieve paginated list.
  • POST /certificate-withdrawal-otherStakeholders/add-formStatus/:user_id: Submit a new verification according to the role.

Request Body Schema

Schema for the request body when adding a form status:


{
    "status": string,
    "authority_id": string,
    "comment": string | null,
    "form_id": number
}

Ensure all fields adhere to the specified data types and formats.
