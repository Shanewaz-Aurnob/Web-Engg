Endpoint: /course-semester

Retrieve information about courses offered in different semesters.

Available Endpoints

1. GET /course-semester/

Retrieve a list of courses offered in various semesters.

  • Query Parameters:
    • page: Page number for pagination (optional)
    • limit: Number of results per page (optional)
    • sort: Sort courses by a specific field (optional)
    • filter: Apply filters to narrow down results (optional)
  • Response:
    • Success (200): Returns a paginated list of courses in semesters.
    • Internal Server Error (500): If there's a server-side issue.

Example Requests

GET /course-semester/

Request:

fetch('http://bike-csecu.com:5000/api/course-semester?page=1&limit=10')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));

Additional Information

The /course-semester endpoint provides a list of courses offered in various academic sessions and semesters. For pagination, use the page and limit query parameters to navigate through the results. You can also sort and filter the courses using the respective query parameters. Contact support for any issues related to the courses in semester endpoint.
