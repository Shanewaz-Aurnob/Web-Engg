Study Leave Evaluation Endpoint

Endpoints to manage study leave evaluations, including retrieval, submission, and update.

Below are the available endpoints:

  • GET /: Retrieve paginated list of study leave evaluations.
  • POST /add: Submit a new study leave evaluation.
  • PUT /update: Update an existing study leave evaluation.

Request Body Schema

Schema for the request body when adding or updating a study leave evaluation:


{
    "applicant_id": string,
    "evaluation_type": string,
    "le_comment": string | null,
    "le_evaluation_time": string (date in ISO format),
    "le_status": string,
    "leave_id": number
}

Ensure all fields adhere to the specified data types and formats.
