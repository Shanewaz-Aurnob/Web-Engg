Exam Activity Endpoint

/exam-activity

Endpoint to manage the exam activities for billing, including retrieval, submission, and update.

Below are the available endpoints:

  • GET /: Retrieve paginated list of exam activities for billing.
  • POST /: Submit a new exam activity for billing.
  • PUT /: Update an existing exam activity for billing.

Example Requests

GET /exam-activity

Request:

fetch('http://bike-csecu.com:5000/api/cuers/exam-activity',{
method: 'GET',
headers: {
    'Authorization': `Bearer ${sessionId}`
    'Content-Type': 'application/json',
    }                
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Error:', error));
