Exam Activity Factors Endpoint

/exam-activity-factors

Endpoint to manage the factors for exam activities for billing, including retrieval, submission, and update.

Below are the available endpoints:

  • GET /: Retrieve paginated list of exam activity factors for billing.
  • POST /: Submit a new exam activity factor for billing.
  • PUT /: Update an existing exam activity factor for billing.

Example Requests

GET /exam-activity-factors

Request:

fetch('http://bike-csecu.com:5000/api/cuers/exam-activity-factors',{
method: 'GET',
headers: {
    'Authorization': `Bearer ${sessionId}`
    'Content-Type': 'application/json',
    }                
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Error:', error));
