Evaluates Exam Endpoint

/evaluates-activity

Endpoint to manage exam activities performed by the teachers, including retrieval, submission, and update.

Below are the available endpoints:

  • GET /: Retrieve exam_id and related data of the teacher participating in exam related tasks.
  • GET /:exam_id: Retrieve factored data of all exam activities performed by the teachers for the specified exam_id.
  • POST /: Submit a new exam activity performed by a teacher.
  • PUT /:activity_id: Update an existing exam activity performed by a teacher.

Example Requests

GET /evaluates-activity

Request:

fetch('http://bike-csecu.com:5000/api/cuers/evaluates-activity',{
method: 'GET',
headers: {
    'Authorization': `Bearer ${sessionId}`
    'Content-Type': 'application/json',
    }                
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Error:', error));
