Bill Calculation Endpoint

/calculate-bill

Endpoint to calculate exam bills for the logged in teacher.

Below are the available endpoints:

  • GET /: Retrieve a list of exams and academic sessions the teacher has participated to incur exam bills.
  • GET /:exam_id/:academic_session_id: Retrieve the teacher's exam bill calculation for the specific exam.

Example Requests

GET /calculate-bill

Request:

fetch('http://bike-csecu.com:5000/api/cuers/calculate-bill',{
method: 'GET',
headers: {
    'Authorization': `Bearer ${sessionId}`
    'Content-Type': 'application/json',
    }                
})
.then(response => response.json())
.then(data => console.log(data))
.catch(error => console.error('Error:', error));

Response:

{
    "data": [
      {
        "exam_id": 4,
        "academic_session_id": 20180801
      }
    ]
  }
