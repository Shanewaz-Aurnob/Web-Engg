Static Resources

Access static resources (For instance: photos, documents, files, etc.) easily. Just request the resource you want and receive a response accordingly.

  • Files: Access uploaded files and resources from this directory directly. You can access a file by its filename. Access URL: 
    /upload/{filename}

    Example: Example File (URL: /upload/example.cpp)
  • Images: Access uploaded images exactly how you'd access files. Access URL: 
    /upload/image/{filename}
    Example: URL: /upload/image/test.webp
